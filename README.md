# 🐟 مشروع أسماك البحار - Asmak Seafood Restaurant

تطبيق ويب متكامل لمطعم أسماك، يعرض القائمة (Menu) والأسعار بشكل حي وجذاب.

## 🚀 المميزات (Features)

- **واجهة مستخدم متجاوبة:** تعمل بكفاءة على الموبايل والكمبيوتر.
- **منيو رقمي:** يعرض كل أصناف الفسفور والأسماك مع تحديث حي للبيانات.
- **تصميم عصري:** استخدام HTML5 و CSS3 لتوفير تجربة مستخدم مريحة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
